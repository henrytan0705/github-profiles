# Github Profiles

- [Live](https://github-profiles-a7571.web.app/) on Firebase.

### GitHub Profile Fetching Application.

#### User's Displayed Data

- Profile Image
- Name
- Bio
- GitHub Username
- Followers
- Public Repos Count
- Github Profile Link
- Blog/Portfolio Link

### Languages and Technologies Used

- Angular
- TypeScript
- Axios
- Bootstrap

### Data Source

- Github API: "https://api.github.com/users/{usernane}"
